# Restaurant Website

## Project Overview

This is a simple restaurant website created using HTML and CSS. The website was designed as part of a practice project to demonstrate basic web development skills.

## Features

* Home page
* Menu section
* About Us section
* Contact Us section
* Responsive and clean design
* Images for menu items and restaurant branding

## Technologies Used

* HTML5
* CSS3

## Project Structure

```
restaurant-website/
│
├── index.html
├── style.css
├── images/
└── README.md
```

## How to Run the Project

1. Download or clone the project files.
2. Open `index.html` in any web browser.
3. Browse the website.

## Purpose

The purpose of this project is to practice creating a simple and attractive website using HTML and CSS without additional frameworks.
